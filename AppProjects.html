Full-stack Ecommerce App

Basit Sanitary App is a modern, cross-platform mobile e-commerce application developed as my Final Year Project. It provides a seamless shopping experience for sanitary products, built specifically for Android and iOS. The app aims to make product discovery and ordering fast, simple, and user-friendly.
The app includes features like product browsing, category-wise search, smart filtering, dynamic cart management, and secure payment methods(e.g Stripe). It also supports user registration/login, personalized user flow, and real-time updates. The UI is intuitive, ensuring a smooth and engaging shopping journey.
The frontend is developed using React Native for cross-platform support, while the backend uses Node.js and Express.js for API handling. MySQL manages all structured data, and Firebase Storage is integrated for scalable image storage. The architecture is designed to support future AI-based enhancements.

Pic Nova AI (AI BAsed Pics & Videos Editing App)

PicNova is an advanced AI-powered photo and video editing application designed for modern users who want creativity with simplicity. With intelligent editing tools, users can enhance their photos, apply filters, generate AI-driven effects, and transform ordinary media into professional-quality visuals within seconds.
The app supports features such as background removal, face enhancement, AI filters, and video stylization, making editing quick, smart, and fun. PicNova is built to serve both casual users and content creators, ensuring powerful editing while keeping the interface intuitive.
The frontend is built using React Native for seamless cross-platform performance, while Node.js and Express.js power the backend. Firebase and cloud-based AI Models are integrated for real-time media processing, ensuring speed, scalability, and creativity at its best.

Tap & Win Game(Lottery type)

Tap & Win Prizes is an exciting lottery-style mobile game designed to bring fun, suspense, and rewards to players. The game features a grid of boxes where some contain hidden prizes while others are empty. Players purchase chances using mobile wallets (e.g., JazzCash, EasyPaisa, or Stripe for global users) and reveal boxes to test their luck.
The app is designed with secure user registration, wallet integration, prize history tracking, and real-time updates, ensuring fair play and engagement. Its smooth animations, rewarding gameplay, and suspense-driven mechanics make it both entertaining and addictive.
Developed with React Native for cross-platform support and Node.js/Express.js for backend APIs along With Sockets for real time user experience, the app uses MySQL for secure user and prize data management.
