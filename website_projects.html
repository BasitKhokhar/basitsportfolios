ShadowSuit (Cybersecurity Project)

ShadowSuite is a premier cybersecurity software platform I developed for an Indian client. The site provides a secure, professional, and user-friendly interface to explore, evaluate, and purchase specialized cybersecurity software.

It is a Next.js project, powers dynamic product interactions, and includes WhatsApp-based client support. The website serves as a commercial hub for elite cybersecurity tools, catering to ethical hackers, red teamers, and researchers. All tools are intended for authorized and educational use, empowering users with cutting-edge, undetectable solutions.

Full-stack Ecommerce Project

Basit Sanitary Store is a fully-functional e-commerce website specializing in a wide range of sanitary products. Built with React.js and styled with Tailwind CSS for a modern and responsive user experience, the website features a seamless interface for browsing, filtering, and purchasing products. The backend, powered by Node.js and MySQL, manages product data, user accounts, and secure transactions, ensuring a robust and efficient database system. This project highlights dynamic cart functionality, user-specific product management, and real-time updates, aimed at providing a reliable shopping platform for customers seeking high-quality sanitary products.
